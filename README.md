# Scentara — Fine Fragrance Website

A luxury perfume brand website for **Scentara**, built with pure HTML, CSS, and vanilla JavaScript. Ready to deploy on GitHub Pages in one click.

## ✨ Features

- Full-page hero with animated perfume bottle SVG
- Scrolling marquee ticker
- Product collection grid (Amber Nuit, Iris Noir, Oud Royale)
- About / Brand story section
- Craft process breakdown
- Newsletter signup
- Custom cursor
- Scroll-triggered reveal animations
- Fully responsive (mobile-friendly)
- Zero dependencies — no frameworks, no build step

## 🚀 Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `scentara`)
2. Upload `index.html` to the root of the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save** — your site will be live at:
   ```
   https://YOUR-USERNAME.github.io/scentara/
   ```

## 🎨 Design

- **Palette**: Warm cream, sand, gold, deep brown
- **Fonts**: Cormorant Garamond (display) + Tenor Sans (body)
- **Style**: Luxury editorial — refined, atmospheric, unforgettable
- **Animations**: Float, fade-up on scroll, marquee, custom cursor

## 📝 Customization

| What to change | Where in `index.html` |
|---|---|
| Brand name | Search `Scentara` |
| Colors | `:root` CSS variables |
| Products & prices | `.product-card` sections |
| Social links | Footer `.social-links` |
| Newsletter form | Connect to Mailchimp / ConvertKit |
